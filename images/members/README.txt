MEMBER PHOTOS FOLDER
====================
Drop member profile photos here.

Naming convention (use member number or name):
  01_cleck_pontillas.jpg
  02_gibe_ibuna.jpg
  03_angelito_dagdag.jpg
  ... and so on

Recommended size: 300x300 px (square), JPG or PNG.

Once added, update the member data in js/members.js — 
change the photo property from '' to the filename, e.g.:
  photo: 'images/members/01_cleck_pontillas.jpg'
